

install
-------

```bash
$ virtualenv --python=python2.7 env
$ env/bin/python setup.py develop
```

self-dist with rpm
------------------

```bash
$ env/bin/rpm_fab
$ ls dist
```


dist egg with rpm
------------------

```bash
$ cd /path/to/your/project-folder
$ /path/to/rpm_fab/env/bin/rpm_fab
$ ls dist
```
